# AI Chat Model

This project is an AI Chat Model built using **LangChain**, **Google Gemini API**, and **Streamlit**.

## Features

- AI-powered chatbot
- Google Gemini API integration
- Built with LangChain
- Simple Streamlit interface
- Real-time responses

## Technologies Used

- Python
- LangChain
- Google Gemini API
- Streamlit

## How to Run

1. Install the required packages:

```bash
pip install -r requirements.txt
```

2. Run the application:

```bash
streamlit run chatmodel.py
```
